# Exercício 05: Dobro, Triplo e Raiz Quadrada

Este programa realiza operações matemáticas básicas a partir de um número inteiro fornecido pelo usuário, calculando seu dobro, seu triplo e sua raiz quadrada.

## Funcionalidades
* Recebe um valor numérico do usuário.
* Calcula o dobro do número (multiplicação por 2).
* Calcula o triplo do número (multiplicação por 3).
* Calcula a raiz quadrada (utilizando exponenciação).
* Exibe os resultados formatados com duas casas decimais.

## Tecnologias Utilizadas
* **Linguagem:** Python
* **Conceitos:** Operadores aritméticos, potenciação (`**`), formatação de números em f-strings (`:.2f`).

## O que aprendi
* Como calcular a raiz quadrada em Python utilizando a potência de $0.5$ ($num^{(1/2)}$).
* A importância da formatação numérica dentro das f-strings para limitar a quantidade de casas decimais em resultados de divisões ou raízes.
