# Exercício 21: Verificador de Par ou Ímpar

Este programa solicita um número inteiro ao usuário e utiliza lógica matemática para determinar se ele é par ou ímpar.

## Funcionalidades
* Recebe um valor inteiro do usuário.
* Aplica a lógica de divisibilidade por 2.
* Exibe uma mensagem formatada e colorida indicando o resultado.

## Tecnologias Utilizadas
* **Linguagem:** Python
* **Bibliotecas:** `rich` (estilização de cores).
* **Conceitos:** Operador de módulo (`%`), estruturas condicionais (`if/else`) e formatação de strings.

## O que aprendi
* Como o operador de módulo (`%`) calcula o resto da divisão. A lógica é simples: se o resto da divisão de um número por 2 for 0, o número é par; caso contrário, é ímpar.
* A versatilidade do operador `%` em diversas situações, como verificar se um ano é bissexto ou criar alternância de cores em linhas de tabelas.
