# Exercício 10: Pintando Parede

Este programa calcula a área total de uma parede a partir de sua largura e altura, e determina a quantidade de tinta necessária para pintá-la, considerando que cada litro de tinta cobre 2 metros quadrados.

## Funcionalidades
* Recebe a largura e a altura da parede do usuário como valores decimais (`float`).
* Calcula a área total da parede através da multiplicação das dimensões.
* Calcula o consumo de tinta com base na área total.
* Exibe os resultados calculados para o usuário.

## Tecnologias Utilizadas
* **Linguagem:** Python
* **Conceitos:** Entrada de dados (`input`), tipos decimais (`float`), operadores aritméticos (`*`, `/`) e formatação de strings (`f-strings`).

## O que aprendi
* A importância de quebrar um problema em etapas menores (primeiro calcular a área, depois o insumo necessário).
* Como utilizar variáveis intermediárias (`area`) para tornar o código mais legível e fácil de manter, ao invés de realizar todos os cálculos dentro do comando `print`.
