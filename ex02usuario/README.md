# Exercício 02: Saudação ao Usuário

Este programa tem como objetivo interagir com o usuário, solicitando seu nome e exibindo uma mensagem de saudação personalizada.

## Funcionalidades
* Solicita a entrada de dados do usuário (nome).
* Exibe uma mensagem de boas-vindas utilizando interpolação de strings.

## Tecnologias Utilizadas
* **Linguagem:** Python
* **Conceitos:** Entrada de dados (`input`), armazenamento em variáveis e formatação de saída (`f-strings`).

## O que aprendi
* Como capturar informações digitadas pelo usuário através do teclado.
* A importância da tipagem e conversão (neste caso, mantendo como string).
* Uso de `f-strings` para criar mensagens dinâmicas de forma legível e eficiente.
