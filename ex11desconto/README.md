# Exercício 11: Calculadora de Descontos

Este programa solicita o preço original de um produto e aplica um desconto fixo de 5%, exibindo o novo valor final para o consumidor.

## Funcionalidades
* Recebe o preço original do produto como um valor decimal (`float`).
* Aplica a lógica de desconto (5% sobre o valor).
* Calcula e exibe o preço com o desconto aplicado.

## Tecnologias Utilizadas
* **Linguagem:** Python
* **Conceitos:** Entrada de dados (`input`), tipos decimais (`float`), operadores aritméticos (`-`, `*`) e formatação (`f-strings`).

## O que aprendi
* A importância da precedência de operadores: como o Python entende que a multiplicação `preco * 0.05` deve ser feita antes da subtração.
* Como criar uma lógica de negócio simples (aplicação de desconto) e integrá-la a uma interface de usuário via console.
