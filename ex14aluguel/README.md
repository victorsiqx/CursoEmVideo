# Exercício 14: Calculadora de Aluguel de Carros

Este programa calcula o valor total a pagar pelo aluguel de um carro, considerando a quantidade de dias utilizados e a quilometragem percorrida pelo usuário.

## Funcionalidades
* Recebe o número de dias alugados (inteiro).
* Recebe a quantidade de quilômetros rodados (inteiro).
* Aplica as taxas fixas: R$ 60,00 por dia e R$ 0,15 por Km rodado.
* Exibe o valor total a ser pago formatado como moeda.

## Tecnologias Utilizadas
* **Linguagem:** Python
* **Conceitos:** Constantes (boas práticas de nomenclatura), entrada de dados (`input`), operadores aritméticos e formatação (`f-strings`).

## O que aprendi
* A importância de definir **constantes** no início do código. Isso facilita a manutenção: se o preço do aluguel ou da quilometragem mudar no futuro, basta alterar o valor no topo do script, sem precisar procurar onde o cálculo é feito.
* Como realizar operações compostas, somando os custos fixos (dias) com os custos variáveis (Km rodados) para obter um valor final.
