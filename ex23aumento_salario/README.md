# Exercício 24: Calculadora de Aumento Salarial Condicional

Este programa calcula o reajuste salarial de um funcionário de forma condicional: funcionários que ganham acima de R$ 1.250,00 recebem um aumento de 10%, enquanto aqueles que ganham R$ 1.250,00 ou menos recebem um aumento de 15%.

## Funcionalidades
* Recebe o salário atual como valor decimal (`float`).
* Aplica a lógica de reajuste utilizando um operador ternário.
* Exibe o novo salário formatado com duas casas decimais.

## Tecnologias Utilizadas
* **Linguagem:** Python
* **Conceitos:** Operadores ternários, estruturas condicionais e formatação de strings.

## O que aprendi
* Como utilizar a estrutura `valor_se_verdadeiro if condição else valor_se_falso` para simplificar blocos `if/else` que atribuem valores a variáveis.
* A importância de planejar as faixas de reajuste para garantir que a regra de negócio da empresa seja aplicada corretamente.
