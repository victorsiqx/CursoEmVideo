# Exercício 07: Conversor de Medidas

Este programa solicita uma distância em metros ao usuário e realiza a conversão para diversas outras unidades de medida (quilômetros, hectômetros, decâmetros, decímetros, centímetros e milímetros).

## Funcionalidades
* Recebe um valor em metros.
* Calcula automaticamente as conversões baseadas nas potências de 10.
* Exibe todos os resultados de forma organizada em uma única saída.

## Tecnologias Utilizadas
* **Linguagem:** Python
* **Conceitos:** Operadores aritméticos, formatação de strings e uso de **docstrings/strings triplas** para textos com quebra de linha.

## O que aprendi
* Como utilizar aspas triplas (`"""`) para criar blocos de texto que respeitam quebras de linha, tornando a saída muito mais legível.
* A aplicação prática da lógica de conversão do Sistema Internacional de Unidades (SI).
