# Exercício 20: Radar Eletrônico

Este programa simula um radar de trânsito. Ele solicita a velocidade de um carro e, caso ela ultrapasse o limite de 80 km/h, calcula o valor da multa a ser paga.

## Funcionalidades
* Recebe a velocidade atual do veículo (inteiro).
* Verifica se o veículo excedeu o limite permitido (80 km/h).
* Calcula o valor da multa, aplicando R$ 7,00 por cada quilômetro acima do limite.
* Exibe mensagens personalizadas com cores diferentes dependendo da situação (multado ou não).

## Tecnologias Utilizadas
* **Linguagem:** Python
* **Bibliotecas:** `rich` (estilização de saída).
* **Conceitos:** Estruturas condicionais (`if/else`), operadores aritméticos (`-`, `*`) e formatação de saída.

## O que aprendi
* Como integrar cálculos matemáticos dentro de uma estrutura condicional.
* A importância de criar "mensagens de feedback" apropriadas para o usuário (cores distintas para situações de perigo ou normalidade).
* O uso de variáveis para armazenar resultados de cálculos que só precisam ser executados sob certas condições.
