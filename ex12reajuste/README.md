# Exercício 12: Calculadora de Aumento Salarial

Este programa calcula o novo salário de um funcionário após a aplicação de um reajuste percentual de 15%.

## Funcionalidades
* Recebe o salário atual do funcionário como um valor decimal (`float`).
* Aplica um aumento percentual definido em uma variável.
* Calcula e exibe o novo salário final formatado com duas casas decimais.

## Tecnologias Utilizadas
* **Linguagem:** Python
* **Conceitos:** Variáveis, entrada de dados (`input`), operadores aritméticos (`+`, `*`) e formatação (`f-strings`).

## O que aprendi
* A importância de criar variáveis como `aumento` para armazenar taxas fixas. Isso torna o código mais flexível: se a empresa decidir mudar o reajuste para 20% no futuro, você só precisa alterar esse valor em um único lugar.
* Como realizar cálculos de incremento percentual de forma legível.
