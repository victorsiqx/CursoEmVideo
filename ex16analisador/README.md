# Exercício 16: Analisador de Textos

Este programa solicita o nome completo do usuário e realiza uma série de processamentos de strings para exibir informações detalhadas sobre o nome digitado.

## Funcionalidades
* Remove espaços desnecessários no início e no final do nome (`strip`).
* Exibe o nome inteiramente em letras maiúsculas e minúsculas.
* Calcula a quantidade de letras totais (desconsiderando os espaços).
* Identifica o primeiro nome e conta quantos caracteres ele possui.

## Tecnologias Utilizadas
* **Linguagem:** Python
* **Conceitos:** Manipulação de strings, métodos de strings (`upper`, `lower`, `strip`, `split`, `count`, `len`).

## O que aprendi
* Como usar o método `strip()` para limpar dados de entrada do usuário.
* A diferença entre contar o tamanho total de uma string (`len`) e contar apenas caracteres específicos (`count`).
* Como transformar uma string em uma lista de palavras usando `split()`, permitindo acessar partes específicas do nome (como o primeiro nome).
