# Exercício: Média Aritmética

Este programa solicita ao usuário que insira duas notas de um aluno e realiza o cálculo da média aritmética entre elas.

## Funcionalidades
* Recebe duas notas como valores decimais (`float`).
* Realiza o cálculo da soma das notas e divide o resultado por dois.
* Exibe a média final calculada.

## Tecnologias Utilizadas
* **Linguagem:** Python
* **Conceitos:** Entrada de dados (`input`), conversão de tipos (`float`), operadores aritméticos (`+`, `/`) e formatação de strings (`f-strings`).

## O que aprendi
* A importância de utilizar o tipo `float` para notas, permitindo valores decimais que não seriam possíveis com `int`.
* A utilização de parênteses para garantir a precedência da soma antes da divisão na fórmula da média.
