# Exercício 32: Contagem Regressiva Explosiva

Este programa executa uma contagem regressiva de 10 a 0 no terminal, utilizando um intervalo de tempo para simular a expectativa antes de uma "explosão" final.

## Funcionalidades
* **Contagem Regressiva:** Utiliza o comando `range` com passo negativo (`-1`) para iterar do maior para o menor valor.
* **Controle de Tempo:** Implementa o método `sleep(0.5)` para pausar a execução entre cada número, tornando a contagem visível ao usuário.
* **Evento Final:** Dispara uma mensagem personalizada ao concluir a contagem.

## Tecnologias Utilizadas
* **Linguagem:** Python
* **Bibliotecas:** `time` (para controle de tempo).
* **Conceitos:** Estruturas de repetição (`for`), intervalos de iteração (`range`) e manipulação de fluxo.

## O que aprendi
* Como funciona a estrutura `range(início, fim, passo)`: ao definir o passo como `-1`, aprendi a controlar o sentido da contagem.
* Como adicionar dinamismo ao terminal através do uso de bibliotecas de tempo, criando uma melhor experiência interativa.
* A estrutura básica de um laço de repetição, onde o código dentro do bloco é executado repetidamente até que a condição do `range` seja satisfeita.
