# Exercício 04: Antecessor e Sucessor

Este programa solicita um número inteiro ao usuário e realiza um cálculo simples para identificar e exibir o seu antecessor e o seu sucessor.

## Funcionalidades
* Recebe um valor numérico do usuário.
* Calcula o antecessor subtraindo 1 e o sucessor somando 1 ao valor original.
* Exibe os resultados de forma clara utilizando f-strings.

## Tecnologias Utilizadas
* **Linguagem:** Python
* **Conceitos:** Operadores aritméticos, entrada de dados (`input`), conversão para inteiro (`int`) e formatação de strings (`f-strings`).

## O que aprendi
* Como realizar cálculos matemáticos dinâmicos diretamente dentro da função `print`, economizando linhas de código.
* A manipulação de números inteiros para operações de incremento e decremento.
