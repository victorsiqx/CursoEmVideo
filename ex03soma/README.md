# Exercício 03: Soma de Dois Valores

Este programa realiza a leitura de dois números inteiros fornecidos pelo usuário e exibe o resultado da soma entre eles de forma formatada.

## Funcionalidades
* Solicita dois valores numéricos inteiros.
* Processa a soma dos valores informados.
* Exibe o resultado final utilizando interpolação de strings.

## Tecnologias Utilizadas
* **Linguagem:** Python
* **Conceitos:** Entrada de dados (`input`), conversão de tipos (`int`), operadores aritméticos (`+`) e `f-strings`.

## O que aprendi
* A necessidade de converter a entrada padrão do Python (que é sempre `string`) para um tipo numérico (`int`) para realizar operações matemáticas.
* Como realizar cálculos diretamente dentro da string formatada, mantendo o código conciso.
