# Exercício 22: Calculadora de Preço de Viagem

Este programa calcula o valor da passagem de uma viagem com base na distância percorrida. Aplica tarifas diferenciadas: viagens mais curtas possuem um custo por quilômetro maior, enquanto viagens mais longas contam com um desconto proporcional.

## Funcionalidades
* Recebe a distância total da viagem (inteiro).
* Aplica a lógica condicional para definir o preço:
    * Até 200 km: R$ 0,50 por km.
    * Acima de 200 km: R$ 0,45 por km.
* Exibe o custo total da viagem formatado.

## Tecnologias Utilizadas
* **Linguagem:** Python
* **Conceitos:** Estruturas condicionais (`if/else`), operadores aritméticos (`*`) e formatação de strings.

## O que aprendi
* A importância de planejar a lógica de negócio antes de codificar: identifiquei as duas faixas de preço e criei variáveis para cada cenário.
* Como utilizar o `if/else` para criar caminhos de execução distintos que alteram o resultado final do programa.
* A diferença entre custos fixos e variáveis, e como aplicar diferentes taxas com base em um critério de entrada (a distância).
