# Exercício 08: Gerador de Tabuada

Este programa solicita um número inteiro ao usuário e gera a sua tabuada completa (de 1 a 10), exibindo o resultado de forma organizada.

## Funcionalidades
* Recebe um número inteiro do usuário.
* Utiliza um loop para iterar de 1 a 10.
* Realiza o cálculo da multiplicação e exibe o resultado formatado.
* Adiciona elementos visuais (traços) para melhorar a legibilidade da saída.

## Tecnologias Utilizadas
* **Linguagem:** Python
* **Conceitos:** Entrada de dados (`input`), conversão para inteiro (`int`), estruturas de repetição (`for` e `range`), operadores aritméticos e formatação (`f-strings`).

## O que aprendi
* A importância do comando `for` para evitar a repetição manual de código.
* Como utilizar a função `range(1, 11)` para definir os limites de uma repetição.
* O uso do operador de multiplicação de strings (`"-" * 20`), um recurso muito útil do Python para criar elementos visuais rapidamente.
