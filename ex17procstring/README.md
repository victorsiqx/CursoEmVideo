# Exercício 17: Verificador de Identidade

Este programa solicita o nome do usuário e verifica, de forma booleana, se a string "Sales" está presente no texto informado.

## Funcionalidades
* Recebe o nome completo do usuário.
* Utiliza o operador de pertinência para buscar a ocorrência de uma substring específica.
* Retorna `True` se o nome contiver "Sales" ou `False` caso contrário.

## Tecnologias Utilizadas
* **Linguagem:** Python
* **Conceitos:** Operadores de pertinência (`in`), manipulação de strings e tipos booleanos.

## O que aprendi
* Como o Python facilita a busca de subconjuntos de caracteres dentro de uma string usando o operador `in`.
* A importância de normalizar dados de entrada (como o uso de `strip()` para evitar que espaços extras causem erros de leitura).
* A eficiência de obter um resultado lógico direto (`True`/`False`) sem precisar de estruturas condicionais complexas para buscas simples.
