# Exercício 09: Conversor de Moeda

Este programa solicita um valor em Reais (R$) ao usuário e realiza a conversão para Dólares (US$), utilizando uma cotação fixa como base de cálculo.

## Funcionalidades
* Recebe um valor monetário como entrada do usuário (`float`).
* Realiza a divisão do valor pelo valor da cotação.
* Exibe o resultado final com a formatação de duas casas decimais, simulando a precisão de centavos.

## Tecnologias Utilizadas
* **Linguagem:** Python
* **Conceitos:** Entrada de dados (`input`), tipos decimais (`float`), operadores aritméticos (`/`) e formatação de saída (`f-strings` com especificador de precisão).

## O que aprendi
* A aplicação prática de variáveis do tipo `float` para lidar com cálculos financeiros, onde casas decimais são indispensáveis.
* Como utilizar o especificador `:.2f` dentro da `f-string` para formatar números, garantindo que o resultado exiba apenas duas casas decimais, o que é fundamental para a leitura correta de valores monetários.
