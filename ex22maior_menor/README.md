# Exercício 23: Identificador de Maior e Menor Valor

Este programa solicita ao usuário que insira três números inteiros, armazena-os em uma lista e, em seguida, identifica automaticamente qual é o maior e o menor valor entre eles.

## Funcionalidades
* Utiliza uma estrutura de repetição (`for`) para coletar três números do usuário.
* Armazena os dados em uma lista dinâmica (`valores`).
* Utiliza funções nativas do Python para processar os dados:
    * `min()`: para encontrar o menor valor.
    * `max()`: para encontrar o maior valor.
* Exibe os resultados para o usuário.

## Tecnologias Utilizadas
* **Linguagem:** Python
* **Conceitos:** Listas (`list`), estruturas de repetição (`for`), métodos de listas (`append`) e funções built-in de Python.

## O que aprendi
* A utilidade das funções `min()` e `max()`, que simplificam tarefas que exigiriam múltiplas estruturas condicionais (if/else) se feitas manualmente.
* Como a estrutura de lista (`append`) em conjunto com o `for` torna o código escalável: se você precisasse de 100 números em vez de 3, bastaria alterar apenas um número no `range()`.
