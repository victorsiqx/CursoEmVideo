# Exercício 25: Comparador de Valores

Este programa solicita dois números inteiros ao usuário e determina a relação de grandeza entre eles (maior, menor ou igual), utilizando uma estrutura de decisão concisa.

## Funcionalidades
* Recebe dois números inteiros do usuário.
* Compara os valores e identifica se o primeiro é maior, se o segundo é maior ou se são equivalentes.
* Exibe o resultado da comparação de forma direta.

## Tecnologias Utilizadas
* **Linguagem:** Python
* **Conceitos:** Operadores ternários aninhados, estruturas condicionais e entrada de dados.

## O que aprendi
* Como encadear operadores ternários (`A if condicao1 else (B if condicao2 else C)`) para realizar múltiplas verificações em uma única linha.
* A importância de manter o código legível: embora o operador ternário aninhado seja conciso, é preciso avaliar se ele não torna a lógica muito complexa para outros desenvolvedores.
* O uso de parênteses para organizar a lógica dentro do operador ternário, garantindo que o Python execute as condições na ordem correta.
