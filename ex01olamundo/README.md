# Exercício 01: Olá Mundo!

Este é o primeiro passo na minha jornada de programação com Python. O objetivo deste exercício é configurar o ambiente de desenvolvimento e executar o comando de saída básico para exibir uma mensagem no console.

## Funcionalidades
* Exibe a clássica mensagem "Olá, Mundo!" na tela.

## Tecnologias Utilizadas
* **Linguagem:** Python

## O que aprendi
* Como configurar o ambiente de desenvolvimento (VS Code).
* Como escrever e executar um script simples em Python utilizando a função `print()`.
