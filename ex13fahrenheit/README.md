# Exercício 13: Conversor de Temperaturas

Este programa realiza a conversão de uma temperatura informada em graus Celsius (ºC) para a escala Fahrenheit (ºF).

## Funcionalidades
* Recebe a temperatura em Celsius como um valor decimal (`float`).
* Aplica a fórmula matemática de conversão: $(C \times 1.8) + 32$.
* Exibe o resultado da conversão com uma casa decimal de precisão.

## Tecnologias Utilizadas
* **Linguagem:** Python
* **Conceitos:** Entrada de dados (`input`), operadores aritméticos (`*`, `+`) e formatação (`f-strings`).

## O que aprendi
* Como traduzir uma fórmula matemática real para a sintaxe do Python.
* A necessidade de respeitar a ordem de precedência das operações matemáticas para obter o resultado correto na conversão.
* Como utilizar o especificador de formatação `:.1f` para controlar a exibição de casas decimais em resultados de conversões.
