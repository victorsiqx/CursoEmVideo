# Exercício 15: Sorteador de Alunos

Este programa solicita o nome de quatro alunos, realiza o sorteio de um deles para ser escolhido e, em seguida, define uma ordem aleatória para as apresentações de todos os participantes.

## Funcionalidades
* Coleta dados de entrada (nomes) através de uma estrutura de repetição (`for`).
* Utiliza a função `choice` para selecionar um item aleatório de uma lista.
* Utiliza a função `shuffle` para embaralhar a ordem dos itens dentro de uma lista.
* Exibe o nome do sorteado e a lista embaralhada com a nova ordem de apresentação.

## Tecnologias Utilizadas
* **Linguagem:** Python
* **Conceitos:** Listas (`list`), estruturas de repetição (`for`), métodos de manipulação de listas (`append`) e o módulo `random`.

## O que aprendi
* A importância de importar módulos externos (`import`) para expandir as funcionalidades básicas do Python.
* A diferença entre `choice` (que escolhe um elemento) e `shuffle` (que altera a ordem original da lista).
* Como utilizar a estrutura `alunos.append(aluno)` para armazenar dados dinamicamente em uma lista.
